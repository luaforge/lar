Overview

LAR is...

LAR is free software and uses the same license as Lua 5.0.

Current version

Current version is 1.0.0 It was developed for Lua 5.0.

Download

LAR can be downloaded from the its Lua Forge page.

What's new

  • [16/jul/2004] Version 1.0.0

Installation

LAR is just a lua script. As such there is no need to install. Just put it in your favorite place and `require "lar"'.
